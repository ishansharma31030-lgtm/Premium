API-First Architecture

Built on the robust C# .NET Revit API. We hook directly into the DocumentChanged event to capture parameter updates the millisecond they happen.

DatumSync_EventHook.cs
public void OnDocumentChanged(object sender, DocumentChangedEventArgs e) 
{
    // Initialize Datum Sync Protocol
    DatumEngine engine = new DatumEngine();
    
    // Capture modified elements instantly
    ICollection<ElementId> modified = e.GetModifiedElementIds();
    
    if (modified.Count > 0) 
    {
        Task.Run(() => 
        {
            engine.PushDeltaToCloud(modified, e.GetDocument());
            Console.WriteLine("Sync Complete. Zero latency.");
        });
    }
}
