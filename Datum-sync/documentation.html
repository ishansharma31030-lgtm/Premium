Getting Started

Installation Guide

Datum Sync installs directly into your Autodesk Revit environment. Follow these steps to install the Add-in and authenticate your workstation.

Revit Version Compatibility

Datum Sync currently supports Autodesk Revit versions 2022, 2023, 2024, 2025, and 2026. Make sure Revit is closed before running the installer.

1. Download the Installer

Download the latest `.msi` package from your account dashboard. If you are an enterprise user, your IT administrator may deploy this silently via Microsoft SCCM.

Silent Install (Admin)
msiexec.exe /i "DatumSync_v2.1.msi" /quiet /qn /norestart

2. Authenticate the Plugin

Once installed, open Revit. Navigate to the new Datum Sync tab in the ribbon and click Initialize. You will be prompted to enter your API key.

appsettings.json 
{
  "DatumSync": {
    "Environment": "Production",
    "AutoSyncInterval": 300,
    "ApiKey": "ds_live_YOUR_API_KEY_HERE"
  }
}
